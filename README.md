# PersonalDeal

**AI-powered compatibility and agreement layer for humans.**

PersonalDeal converts human intentions, preferences and constraints into structured profiles and allows instant comparison between people before interaction.

---

## Problem

Human interactions are inefficient:

* deals fail due to hidden expectations
* teams break due to mismatched values
* time is wasted on incompatible people

There is no system to **compare humans before interaction**.

---

## Solution

PersonalDeal introduces:

* AI-generated personal profiles
* QR-based sharing
* real-time compatibility analysis
* conflict detection
* agreement generation

---

## How it works

1. User completes AI-driven questionnaire
2. System builds structured profile
3. Profile is shared via QR or link
4. Another user scans or matches online
5. System compares both profiles
6. Outputs:

   * compatibility score
   * conflicts
   * red flags
   * suggested agreement

---

## Modes

### 1. QR Mode

Scan → compare → decide

### 2. Online Matching

System continuously finds:

* Top 10 compatible users
* ranked by compatibility
